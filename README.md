## Deploy
1) Upload `index.html`, `favicon.ico`, `favicon.png` to your `*.github.io` repo **root**.
2) Add a blank `.nojekyll` at the repo root (no extension).
3) Settings → Pages → Deploy from `main` & **root**.
4) Open your site and hard refresh.
